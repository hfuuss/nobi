# Nobi
### 纪念 Nobi哒哒哒
Nobi哒哒哒: 马美，喜欢《哆啦A梦》动画片。妖性十足的一个小姑凉。        
Nobi Nobita: 野比大雄，直译即野比伸太或野比野比太，中文通称大雄，旧译叶大雄、野比太、康夫、大宝等，英/美国版本称Noby。是藤子·F·不二雄的漫画和动画作品《哆啦A梦》中的主人公，也是作品首位出场角色。
